# WebLogic | Infraestructura Digital y Automatización

Plataforma de ingeniería de software y automatización con IA para negocios.

## Stack Tecnológico

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Framer Motion
- **Language:** TypeScript
- **Deployment:** Vercel

## Desarrollo Local

1. Instalar dependencias:
```bash
npm install
```

2. Iniciar servidor de desarrollo:
```bash
npm run dev
```

3. Abrir [http://localhost:3000](http://localhost:3000)
